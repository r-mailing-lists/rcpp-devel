# rcpp-devel

Rcpp development

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://lists.r-forge.r-project.org/pipermail/rcpp-devel/](https://lists.r-forge.r-project.org/pipermail/rcpp-devel/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics

## Viewer

Browse these archives at [r-mailing-lists.github.io/viewer](https://r-mailing-lists.github.io/viewer/lists/rcpp-devel/)
